# Shared Jsonnet code for @s0cks/dotfiles

This folder contains shared Jsonnet used when installing these dotfiles


| File    | Description |
|:-       |:-           |
| [flexoki.libsonnet](flexoki.libsonnet) | Contains useful variables and functions for generating Flexoki compliant themes            |
| [plist.libsonnet](plist.libsonnet)     | Contains useful code for generating .pist files - Like thos used in launchd (macos)        |
| [util.libsonnet](util.libsonnet)       | Contains some generic utilities to use when geneating files. Like generating file comments |
| [zsh.libsonnet](zsh.libsonnet)         | Contains some utilities for generating zsh scripts                                         |
